| Étape | Commande / Action                    | Objectif                    |
| :---- | :----------------------------------- | :-------------------------- |
| 1     | `ssh-keygen -t ed25519 -C "email"`   | Créer la paire de clés      |
| 2     | `cat ~/.ssh/id_ed25519.pub`          | Copier la clé publique      |
| 3     | Paramètres GitHub → SSH and GPG keys | Ajouter la clé publique     |
| 4     | `ssh -T git@github.com`              | Vérifier la connexion       |
| 5     | `ssh-add ~/.ssh/id_ed25519`          | Charger la clé dans l’agent |
